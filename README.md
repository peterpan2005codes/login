# Demo Login UI

This repository contains a **Demo Login UI** built for demonstration and learning purposes.

## Overview

- Simple and clean login screen design  
- Focused on UI layout and styling  
- No real authentication or backend integration  

## Purpose

This project is intended to:
- Showcase a basic login interface
- Practice UI design and layout structure
- Serve as a starting point for further development

## Note

This is a demo project only and should not be used in production without proper security and backend implementation.
